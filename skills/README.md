# Agent Skills

## Use

### Clone the Repository

You should clone this repository directly into your global skill directory for your agent of choice.

In the case of Codex, this is _~/.codex/skills_.

For example:
```bash
git clone https://github.com/giftbit/agent-skills.git ~/.codex/skills
```

Ensure you update the _.gitignore_ accordingly to avoid committing any skill files you don't wish to include in this repo.

### Update your AGENTS.md

To encourage the agents to actually use your skills, particularly when you're not explicitly saying "use skill X", you should update your AGENTS.md files to include directives on when to use the skills.

See the [EXAMPLE_AGENTS.md](EXAMPLE_AGENTS.md) file for more details.
